=============================================
TROUBLESHOOTING
=============================================

Problem: "Database connection failed"
Solution:
   • Check config/database.php credentials
   • Verify database name: u314077991_adiari_shop
   • Ensure database user has permissions

Problem: "Page not found" for all pages
Solution:
   • Check .htaccess uploaded correctly
   • Verify mod_rewrite enabled (Hostinger has it)
   • Check index.php in root

Problem: "Internal Server Error"
Solution:
   • Check logs/php_errors.log
   • Verify PHP version (≥ 7.4)
   • Check file permissions

Problem: Products not showing
Solution:
   • Verify database import completed
   • Check sample products in database
   • Clear any cache
